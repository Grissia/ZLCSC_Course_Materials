# Docs

這個目錄放的是課堂補充教材，包含：

- 範例程式
- 自主練習挑戰
- 相關筆記

## 檔案結構

- `reverse1/`
- `reverse2/`
- `pwn1/`
- `pwn2/`
- `pwn3/`

## 如何使用

1. 先進入對應章節目錄（例如 `reverse1/`）。
2. 依課堂進度閱讀或執行範例。
3. 搭配下方工具文件查詢指令與系統呼叫細節。
